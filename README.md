## Description

Prueba tecnica Macropay usando NestJs.

## Installation
Yarn
```bash
$ yarn install
```
Npm
```bash
$ npm install
```

## Running the app
Yarn 
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
Npm
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
